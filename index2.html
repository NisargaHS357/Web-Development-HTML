📦 Understanding the CSS Box Model Used Here

Box Model Property Where We Used It What It Does
Content .product-card width: 280px Sets the content area width of each product card
Padding .product-card padding: 20px Creates space INSIDE each card between content and border
Border .product-card border: 1px solid #e0e0e0 Draws a visible edge around each product card
Margin .products-grid gap: 30px Creates space OUTSIDE cards between them
box-sizing * { box-sizing: border-box; } Makes padding and border included in width calculation
